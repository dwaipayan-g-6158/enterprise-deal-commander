# Product Documents

The source-of-truth product materials for Enterprise Deal Commander. These are the *intent*
documents; for what has actually shipped, see [../roadmap.md](../roadmap.md).

## Product Requirement Documents (PRDs)

| Document | Edition | Summary |
|---|---|---|
| [EDC-Phase-1-PRD.md](./EDC-Phase-1-PRD.md) | **Executive War Room Edition** | The correct, deterministic, self-contained deal recorder and risk cockpit. Contains the Phase Boundary Charter, glossary, tech stack, the risk-pattern registry, and features F1–F14. |
| [EDC-Phase-2-PRD.md](./EDC-Phase-2-PRD.md) | **Sovereign Intelligence Edition** | Predictive scoring, cohort analytics, persisted history, Deal Memory, automation, and multi-commander collaboration. |
